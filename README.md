# SMPLComp - Simple Compressor

## Introduction
This is a simplified version of the CTAGDRC plugin.

## Features
- Input Gain
- Threshold/Ratio/Knee
- Attack/Release
- Makeup
- Mix
- Gainreduction/Input/Output Metering
